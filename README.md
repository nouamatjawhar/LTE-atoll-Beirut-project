# LTE Network Planning – Beirut (Atoll Simulation)

## Project Overview
This project presents an LTE cellular network planning and simulation using Atoll software for selected sites in Beirut. The goal is to analyze coverage, signal quality, and network performance for a real-world deployment scenario.

## Objectives
- Design LTE network topology for Beirut area
- Perform RF planning and propagation analysis
- Evaluate coverage (RSRP), quality (SINR), and throughput
- Optimize site placement and antenna parameters

## Tools Used
- Atoll Radio Planning Software
- Excel (Link Budget Analysis)
- GIS Data (Map layers)
- LTE technology (3GPP standards)

## Key Outputs
- Coverage maps (RSRP, SINR)
- Link budget calculations
- Site configuration analysis
- Performance evaluation

## Results Summary
- Good coverage achieved in urban zones
- Edge performance depends on terrain and clutter
- Optimization improves SINR by adjusting antenna tilt and power
